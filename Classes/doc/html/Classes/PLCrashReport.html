  machExceptionInfo

Mach exception information, if available. This will only be included in the case that encoding crash reporter’s exception-based reporting was enabled, and a Mach exception was caught.

@property (nonatomic, readonly) PLCrashReportMachExceptionInfo *machExceptionInfo

Discussion

Mach exception information, if available. This will only be included in the case that encoding crash reporter’s exception-based reporting was enabled, and a Mach exception was caught.

Warning: If Mach exception information is available, the legacy signalInfo property will also be provided; this s required to maintain backwards compatibility with the established API. Note, however, that the signal info may be derived from the Mach exception info by the encoding crash reporter, and thus may not exactly match the kernel exception-to-signal mappings implemented in xnu. As such, when Mach exception info is available, its use should be preferred.
